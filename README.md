# Maglo - Finansal Takip Platformu

Masaüstü öncelikli finansal takip platformu.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Teknolojiler

- React 18
- TypeScript
- Vite
- React Router
- Zustand
- TanStack React Query
- Tailwind CSS
- React Hook Form + Zod
- Recharts
- React Hot Toast
